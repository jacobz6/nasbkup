## 1. 产品概述

NAS Backup 是一个 NAS 文件冷备份管理系统，将 NAS 上的文件备份到阿里云 OSS 冷存储，支持增量备份、去重、压缩、加密等核心功能。前端为管理控制台，面向 NAS 管理员，提供备份状态监控、内容配置、策略管理和日志审计能力。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 直接访问 | 全部功能：监控、配置、触发备份、查看日志 |

### 2.2 功能模块

1. **全览页面**：备份状态概览、容量仪表盘、快捷统计卡片、操作按钮
2. **内容选择页面**：备份目录管理、文件大小限制、排除规则管理
3. **策略设置页面**：定时任务配置、压缩级别、上传设置、保留策略、加密配置、删除策略
4. **日志页面**：备份日志查看、级别过滤、关键词搜索、时间范围筛选

## 3. 核心流程

### 3.1 备份触发流程

用户在全览页面点击"全量备份"或"增量备份"按钮 → 前端调用 POST /api/backup/trigger → 后端返回 202（备份ID和pending状态）→ 前端轮询 GET /api/backup/status 获取实时状态 → 备份完成或失败后停止轮询 → 刷新仪表盘数据

### 3.2 配置修改流程

用户在策略页面修改配置 → 前端调用 PUT /api/strategy/* → 后端更新数据库 config_kv 表并同步运行时配置 → 返回更新后的配置 → 前端显示保存成功

### 3.3 日志查询流程

用户在日志页面选择过滤条件 → 前端调用 GET /api/logs（带查询参数）→ 后端返回分页日志列表 → 前端渲染日志表格 → 用户翻页或修改筛选条件时重新请求

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝灰（#1e293b）为底，冰蓝（#38bdf8）为强调色，搭配暗色系仪表盘风格
- **辅助色**：翠绿（#34d399）表示成功/完成，琥珀（#fbbf24）表示警告，玫红（#f43f5e）表示错误
- **按钮风格**：圆角（8px），主要按钮带微妙渐变，次要按钮为描边风格
- **字体**：标题使用 JetBrains Mono（等宽技术感），正文使用 Noto Sans SC（中文优化）
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式布局，深色主题
- **图标风格**：使用 Lucide 图标库，线条风格，2px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 全览页面 | 状态卡片 | 顶部横幅，渐变背景，大字状态标签，脉冲动画表示运行中 |
| 全览页面 | 容量仪表 | SVG 环形进度条，中心显示百分比，下方图例 |
| 全览页面 | 统计卡片 | 4列网格，每卡片含图标、数值、标签，hover 微上浮 |
| 全览页面 | 操作按钮 | 按钮组，主按钮蓝色渐变，危险按钮红色描边 |
| 全览页面 | 备份历史 | 表格，状态列带颜色标签，分页器 |
| 内容选择页面 | 备份目录管理 | 表格 + 右侧滑出面板编辑，启用/禁用开关 |
| 内容选择页面 | 排除规则管理 | 表格 + 右侧滑出面板编辑，类型下拉选择，启用/禁用开关 |

| 日志页面 | 日志列表 | 表格，级别列带颜色标签，消息列可展开详情，顶部筛选栏 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px
- 导航栏在窄屏下折叠为图标模式
- 统计卡片在中等屏幕下从4列变为2列
- 表格在窄屏下支持横向滚动

### 4.4 动效设计

- 页面切换：淡入淡出（200ms）
- 卡片加载：骨架屏占位
- 备份运行中：状态指示器脉冲动画
- 按钮交互：hover 上浮 + 阴影加深
- 数据更新：数值变化时平滑过渡
